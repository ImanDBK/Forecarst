# Forecast
### An ML predictor app for car prices in sg

## Features

### Backend
- Scrape the web for car data
- Add data into PostgreSQL
- use scikit-learn for ML prediction
- Reminders for users for their price predictions
- APIs using FAST.API

### Frontend
- Usable Web UI using Next.js
- Dashboard for Users
- Search bar eg. "I would like to buy a car for $x in 6 months"
